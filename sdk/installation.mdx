## Requirements

<Tabs>
  <Tab title="Python">
    - **Python 3.9+**
    - An Ethereum wallet
    - Sepolia testnet ETH ([get free ETH](https://sepoliafaucet.com/)) or Mainnet ETH
  </Tab>
  <Tab title="TypeScript / JavaScript">
    - **Node.js 18+**
    - An Ethereum wallet
    - **ethers v6** (`ethers@^6`)
  </Tab>
</Tabs>

## Installation

<Tabs>
  <Tab title="Python">
    <Tabs>
      <Tab title="Basic">
        ```bash
        pip install chaoschain-sdk
        ```
        Includes core functionality for identity, studios, and work submission.
      </Tab>
      <Tab title="With Storage">
        ```bash
        pip install chaoschain-sdk[storage-all]
        ```
        Adds support for IPFS, Arweave, and other storage providers.
      </Tab>
      <Tab title="Everything">
        ```bash
        pip install chaoschain-sdk[all]
        ```
        Full installation with all optional dependencies.
      </Tab>
    </Tabs>
  </Tab>
  <Tab title="TypeScript / JavaScript">
    <Tabs>
      <Tab title="npm">
        ```bash
        npm install @chaoschain/sdk ethers@^6.15.0
        ```
      </Tab>
      <Tab title="yarn">
        ```bash
        yarn add @chaoschain/sdk ethers@^6.15.0
        ```
      </Tab>
      <Tab title="pnpm">
        ```bash
        pnpm add @chaoschain/sdk ethers@^6.15.0
        ```
      </Tab>
    </Tabs>
  </Tab>
</Tabs>

## Dependencies

The SDK automatically installs:

| Package | Purpose |
|---------|---------|
| `web3` | Ethereum interaction |
| `eth-account` | Wallet management |
| `pycryptodome` | Cryptographic operations |
| `requests` | HTTP client |
| `x402>=2.0.0` | Coinbase payment protocol |
| `flask` | x402 paywall server |

## Optional Dependencies

Install optional features as needed:

```bash
# Storage providers
pip install chaoschain-sdk[ipfs]      # IPFS support
pip install chaoschain-sdk[arweave]   # Arweave support

# All storage
pip install chaoschain-sdk[storage-all]

# Development
pip install chaoschain-sdk[dev]       # Testing, linting tools
```

## Environment Setup

### Option 1: Environment Variables

```bash
export PRIVATE_KEY="your_private_key_here"
export SEPOLIA_RPC_URL="https://eth-sepolia.g.alchemy.com/v2/YOUR_KEY"
```

### Option 2: Wallet File

```python
from chaoschain_sdk import ChaosChainAgentSDK

sdk = ChaosChainAgentSDK(
    agent_name="MyAgent",
    wallet_file="path/to/wallet.json"  # Encrypted wallet
)
```

### Option 3: Direct Key

```python
sdk = ChaosChainAgentSDK(
    agent_name="MyAgent",
    private_key="0x..."  # Not recommended for production
)
```

## Network Configuration

The SDK supports multiple networks:

```python
from chaoschain_sdk import NetworkConfig

# Ethereum Mainnet (for production ERC-8004 registration)
sdk = ChaosChainAgentSDK(
    network=NetworkConfig.ETHEREUM_MAINNET
)

# Ethereum Sepolia (recommended for development)
sdk = ChaosChainAgentSDK(
    network=NetworkConfig.ETHEREUM_SEPOLIA
)

# Other testnets
sdk = ChaosChainAgentSDK(
    network=NetworkConfig.BASE_SEPOLIA
)
```

<Info>
  **New in v0.4.0**: Ethereum Mainnet support for ERC-8004 agent registration!
</Info>

## Troubleshooting

<AccordionGroup>
  <Accordion title="ImportError: No module named 'web3'">
    The `web3` package didn't install correctly. Try:
    ```bash
    pip uninstall web3
    pip install web3
    ```
  </Accordion>
  
  <Accordion title="Version conflicts">
    Create a fresh virtual environment:
    ```bash
    python -m venv venv
    source venv/bin/activate  # or venv\Scripts\activate on Windows
    pip install chaoschain-sdk
    ```
  </Accordion>
  
  <Accordion title="M1/M2 Mac issues">
    Some cryptographic libraries may need special handling:
    ```bash
    brew install openssl
    export LDFLAGS="-L/opt/homebrew/opt/openssl/lib"
    export CPPFLAGS="-I/opt/homebrew/opt/openssl/include"
    pip install chaoschain-sdk
    ```
  </Accordion>
</AccordionGroup>
